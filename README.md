# Gorsel Analiz - Yapay Zeka Destekli Gorsel Analiz Platformu

Kullanicilarin yukledigi gorselleri yapay zeka ile analiz eden ve aciklayan web uygulamasi.

## Ozellikler
- Gorsel yukleme ve yapay zeka analizi
- Gorsel gecmisi (onceki analizlere erisim)
- Prisma ORM ile veritabani yonetimi
- Kullanici verisi ve gecmis kayitlarinin depolanmasi
